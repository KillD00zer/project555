## 4. النماذج الإحصائية المختارة

### النموذج الأول: One-Way ANOVA
**من:** المحاضرة الثالثة (Lecture 3)

| البند | التفاصيل |
|---|---|
| **السؤال البحثي** | هل يختلف متوسط مستوى الاكتئاب (Q18) باختلاف وقت الاستخدام اليومي للسوشيال ميديا (Q8)؟ |
| **المتغير المستقل (IV)** | Q8 — وقت الاستخدام اليومي (5 فئات ترتيبية) |
| **المتغير التابع (DV)** | Q18 — مستوى الاكتئاب (مقياس 1-5) |
| **الفرضية الصفرية H₀** | µ₁ = µ₂ = µ₃ = µ₄ = µ₅ (لا يوجد فرق بين المجموعات) |
| **الفرضية البديلة H₁** | على الأقل مجموعتان تختلفان في المتوسط |
| **اختبارات مسبقة** | Shapiro-Wilk (الطبيعية) + Levene's Test (تجانس التباينات) |
| **مقارنات بعدية** | Tukey HSD Post Hoc (لو رفضنا H₀) |

**مبرر الاختيار:** Q8 فئوي بـ 5 مجموعات → لا يصح T-test → يُستخدم ANOVA

---

### النموذج الثاني: MANOVA
**من:** المحاضرة الرابعة (Lecture 4)

| البند | التفاصيل |
|---|---|
| **السؤال البحثي** | هل يختلف النمط النفسي الكلي (اكتئاب + نوم + قلق) باختلاف الجنس؟ |
| **المتغير المستقل (IV)** | Q2 — Gender (Male / Female) |
| **المتغيرات التابعة (DVs)** | Q18 (الاكتئاب) + Q20 (النوم) + Q13 (القلق) |
| **الفرضية الصفرية H₀** | لا يوجد فرق في Centroid المتغيرات التابعة بين الجنسين |
| **الفرضية البديلة H₁** | يوجد فرق في الـ Centroid |
| **اختبارات مسبقة** | Box's M Test (تجانس مصفوفات التباين) |
| **اختبار التأثير الكلي** | Wilks' Lambda |
| **اختبارات أحادية** | Univariate Tests لكل DV على حدة + Levene's Test |

**مبرر الاختيار:** عندنا أكثر من DV واحد مترابطة → MANOVA أدق من عمل 3 ANOVAs منفصلة لتجنب Type I Error

---

### النموذج الثالث: Factor Analysis (PCA)
**من:** Lecture 2 — Factor Analysis (pptx)

| البند | التفاصيل |
|---|---|
| **الهدف** | تجميع الـ 12 سؤال (Q9→Q20) في عوامل نفسية أساسية |
| **المتغيرات** | Q9, Q10, Q11, Q12, Q13, Q14, Q15, Q16, Q17, Q18, Q19, Q20 |
| **طريقة الاستخراج** | Principal Component Analysis (PCA) |
| **اختبار الملاءمة** | KMO (يجب ≥ 0.6) + Bartlett's Test of Sphericity (يجب p < 0.05) |
| **تحديد العوامل** | Eigenvalue > 1 + Scree Plot |
| **دوران العوامل** | Varimax Rotation (Orthogonal) |
| **المخرجات المتوقعة** | 2-3 عوامل نفسية رئيسية: (عامل الإدمان، عامل الاكتئاب، عامل القلق) |

**مبرر الاختيار:** الداتا تحتوي على 12 سؤال Likert مترابطة منطقياً → Factor Analysis يكشف البنية الكامنة ويُبسّط التحليل
